# 구독박스 (Subbox)

AI 툴, OTT, 음악 등 모든 구독 서비스를 한 곳에서 관리하고, 종료일 전 알림을 받는 구독 관리 웹서비스

## 프로젝트 정보
- **유형**: 도구 (SaaS-lite)
- **타겟 사용자**: 여러 구독을 쓰는 개인 / 소규모 팀·스타트업
- **출시 목표**: 1개월 이내

## 핵심 기능
- 구독 서비스 등록 (이름, 가격, 종료일, 카테고리)
- 전체 구독 현황 대시보드 (월 총 지출 한눈에 보기)
- 종료일 D-3, D-1 알림 (브라우저 푸시 + 카카오톡)
- 로그인 없이 로컬 저장 (브라우저 기반)
- 구독 카테고리 필터 (AI, OTT, 음악, 기타)

## 폴더 구조 (5개 팀)
- `planning/` — 기획 팀: 무엇을 만들 것인가
- `design/` — 디자인 팀: 어떻게 보여줄 것인가
- `cx/` — CX 팀: 누가 어떻게 사용할 것인가
- `marketing/` — 마케팅 팀: 어떻게 알릴 것인가
- `strategy/` — 전략 팀: 어떻게 지속 가능하게 만들 것인가
