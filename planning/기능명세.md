# 기능 명세 (Feature Specification)

## 1. 구독 등록

| 항목 | 내용 |
|------|------|
| 서비스명 | 텍스트 입력 (예: ChatGPT Plus, Netflix) |
| 카테고리 | AI툴 / OTT / 음악 / 업무툴 / 기타 선택 |
| 월 구독료 | 숫자 입력 + 통화 선택 (KRW/USD) |
| 결제 주기 | 월간 / 연간 선택 |
| 시작일 | 날짜 선택기 |
| 종료일 (갱신일) | 날짜 선택기 |
| 메모 | 선택 입력 (예: 가족 공유 중, 팀 계정) |

---

## 2. 대시보드

- 전체 구독 카드 목록 표시
- 월 총 지출 합산 금액 (상단 고정)
- 종료일 임박 순 정렬 옵션
- 카테고리별 필터 탭
- D-30 이하: 주황 강조 / D-7 이하: 빨강 강조

---

## 3. 알림 시스템

### 브라우저 푸시 알림
- 종료 3일 전 (D-3) 오전 10시 발송
- 종료 당일 (D-day) 오전 9시 발송
- 브라우저 알림 허용 시 자동 등록

### 카카오톡 알림
- 카카오 알림톡 API 연동
- 전화번호 등록 시 활성화
- 발송 시점: 브라우저 푸시와 동일

---

## 4. 구독 관리 (수정/삭제)

- 카드 클릭 시 수정 모달 열림
- 삭제 시 확인 팝업 노출
- 종료된 구독 아카이브 처리 (삭제 아닌 보관)

---

## 5. 데이터 저장

- localStorage 기반 저장 (로그인 불필요)
- JSON 내보내기 / 가져오기 기능 (백업용)

---

|------|:---:|

| 카테고리 필터 | ✅ |
| 브라우저 푸시 알림 | ✅ |
| 카카오톡 알림 | ❌ (v2) |
