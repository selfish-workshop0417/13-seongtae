# 비주얼 컨셉

## 톤앤매너

- **키워드**: 깔끔함, 신뢰감, 정보 집약
- **레퍼런스**: Linear, Notion, Raycast
- **무드**: 다크모드 기본 / 라이트모드 선택 가능

---

## 컬러 시스템

| 역할 | 색상 | 용도 |
|------|------|------|
| Primary | `#6366F1` (인디고) | 버튼, 강조 |
| Success | `#22C55E` (그린) | 정상 구독 |
| Warning | `#F97316` (오렌지) | D-30 이하 |
| Danger | `#EF4444` (레드) | D-7 이하, D-day |
| Background | `#0F0F13` | 다크 배경 |
| Card | `#1A1A24` | 구독 카드 배경 |
| Text | `#F1F1F5` | 본문 텍스트 |

---

## 타이포그래피

- **폰트**: Pretendard (한국어), Inter (영문)
- **월 총지출 금액**: 32px Bold — 가장 눈에 띄게
- **구독 서비스명**: 16px SemiBold
- **부가 정보 (종료일, 금액)**: 14px Regular, 회색 처리

---

## 화면 흐름

```
[대시보드]
  ├── 상단: 월 총지출 합산 + D-day 임박 배너
  ├── 중단: 카테고리 필터 탭 (전체/AI/OTT/음악/업무툴/기타)
  └── 하단: 구독 카드 그리드 (2열)
       │
       ├── 카드 클릭 → [수정 모달]
       └── + 버튼 → [구독 추가 모달]
                        └── 입력 완료 → 대시보드로 복귀
```

---

## 구독 카드 구성

```
┌─────────────────────────────┐
│ 🎬 Netflix          OTT     │
│ ₩17,000 / 월               │
│ 종료일: 2026.04.24  ⚠️ D-7  │
└─────────────────────────────┘
```

- 서비스 로고 or 이모지 아이콘
- 카테고리 뱃지 (우상단)
- 월 구독료
- 종료일 + D-day 카운트다운
- 임박도에 따라 카드 테두리 색상 변화
